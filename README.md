# DistShift(Minigrid)_Q-Learning

This repository contains a Python implementation of the **Tabular Q-Learning** algorithm designed to solve the `MiniGrid-DistShift` environment. 

While navigating an empty room is straightforward, the `DistShift` environment introduces a critical challenge: **lava**. The agent must learn a safe policy to reach the green goal square without stepping into the hazardous lava tiles, which immediately terminate the episode.
